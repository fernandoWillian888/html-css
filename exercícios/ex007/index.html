Principais formatações

m2 é usado para medir área, para dar esse significado e efeito visual (apesar de não ser o foco) ao 2 utilizamos a tag <sup>

x2 o 2 representa a quantidade de moléculas do elemento x para dar esse significado e efeito visual (apesar de não ser o foco) ao 2 utilizamos a tag <sub>

para mostrar um texto que deve ser lido porém não considerado utilize a tag <del>

Para dar ênfase a um trecho do texto ou destacar um treccho inserido utilize a tag <ins>

Para deixar um trecho técnico ou algo do tipo em itálico utilize a tag <em>

Quando quiser marcar (destacar) um trecho muito importante utilize a tag <mark>

Para destacar um trecho de texto com maior importância utilize a tag <strong>

Para inserir trechos com menos importância utilize a tag <small>

Para fazer uma citação dentro do parágrafo Muito risco, pouco egoComo fiz agora utilize a tag <q>
